# Kaohsiung
利用laravel 5.1+jQuery製作一個全方位建築智慧化管理系統
## 主要功能
* 旅運事物中心模組、公告消息、物料管理模組..等等
* 郵件管理模組、寄放管理模組、失物招領模組...等等
* 保全巡檢模組、清潔檢查模組...等等
* 機電設備管理模組、智慧化應用設備模組、修繕故障通報模組...等等
* 帳號管理模組、樓層管理模組、平面圖模組...等等
